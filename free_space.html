ai输入补全插件

大模型最擅长预测下一个词，输入法也是。这是一个ai输入法插件，调用视觉模型看到你的屏幕和输入，在你输入一个词后自动调用，生成后续输入。

c盘清理工具

智能 Windows C 盘空间清理与分析工具。除了常规的系统清理，还能接入大模型告诉你每个文件夹的作用、风险等级和清理建议。

文件转换工具

本地离线文件转换工具。完全免费、开源，所有转换过程都在本地离线完成，没有文件大小限制，也没有隐私泄露风险。

LLM 文档智能填充

基于 LLM 的 Word 文档自动填充工具。读取模板识别占位符，根据提供的字段数据，利用 LLM 智能生成和填充内容。

智能文档生成工具

基于 Streamlit 和 LLM 的智能文档生成工具。根据 Word 模板和自然语言要求，自动生成格式一致、内容符合需求的新文档。

Use AI Every Server

在本地使用AI编程，自动同步代码到远程服务器并执行，流式输出运行日志。让你在老旧服务器上也能用AI写代码。
